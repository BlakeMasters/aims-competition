# Unused Submissions

This directory contains alternate competition submission ZIPs produced during the search process.

These files are included for auditability and comparison, but they are not the selected final submission. The final selected upload remains:

```text
submission/submission_domain_live_mid600.zip
```

The accompanying `manifest.csv` lists file names and sizes for the archived alternate ZIPs.

Publication notes:

- The directory intentionally contains ZIP artifacts only, not validation extraction directories or Python caches.
- The final selected ZIP is excluded from this directory to avoid ambiguity.
- Hosted outcomes are summarized in `results/eval_finals.txt` and in the report tables.
