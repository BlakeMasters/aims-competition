# Technical Report

Primary artifact:

- `main.pdf`

Source files:

- `main.tex`
- `references.bib`
- `figures/`
- `tables/`

The report is written for an outside competition evaluator. It describes the method families tried using equations and plain-language method descriptions rather than relying on access to the original exploratory code.

To rebuild:

```powershell
tectonic main.tex
```

The generated PDF uses hosted results from `../results/eval_finals.txt` and the derived CSV tables included in this repository.
